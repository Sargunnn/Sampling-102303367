Sampling Techniques and Model Accuracy Comparison

This project analyzes how different sampling techniques affect the accuracy of machine learning models using the Credit Card Fraud Detection Dataset.

Sampling Methods Used

1. Simple Random Sampling
2. Systematic Sampling
3. Stratified Sampling
4. Cluster Sampling
5. Bootstrap Sampling

Models Used

1. Logistic Regression
2. K-Nearest Neighbors
3. Random Forest
4. Support Vector Machine
5. Gradient Boosting

Objective

To determine which sampling technique gives the highest accuracy for each model.

Tools

Python, Pandas, NumPy
